# 飞书卡片 Footer 规范

## 设计原则

- 流式输出过程中：只显示单轮 footer（token/缓存/上下文 + 状态/耗时/模型）
- 输出完成后：显示完整统计（会话/今日/本月 + 单轮 footer）
- 行间距与正常文字保持一致，不要额外增大

## 数据格式说明

### Token 统计
- 格式：`🪙 input+output`
- 单位：小于 10000 显示原始数字，否则显示 `x.xxw`

### 缓存命中率
- 格式：`⚡ rate%`
- 公式：`cacheRead / (cacheRead + cacheWrite + inputTokens) * 100`
- 上限：100%

### 上下文使用
- 格式：`🧠 used/total (pct%)`
- `totalTokens`：当前使用的 token 总数
- `contextTokens`：模型上下文窗口大小

### 会话统计（仅终态显示）
- 会话：`💬 会话 N 轮 · 🪙 total · ⚡ rate%`
- 今日：`📅 今日 N 轮 · 🪙 total · ⚡ rate%`
- 本月：`📆 本月 N 轮 · 🪙 total · ⚡ rate%`

## 停止按钮

- 流式输出过程中显示 `⏹️ 停止` 按钮
- 点击后立即停止 LLM 生成
- 终态卡片自动移除停止按钮
